# 💸 Tip Calculator (C++)

A simple yet powerful C++ console application that calculates tips and splits bills — just like real-world payment apps.

---

## 🚀 Features

* 📋 Interactive tip menu (15%, 20%, 25%, or custom)
* 💰 Calculates tip amount based on bill
* 🧾 Displays total bill with tip
* 👯 Splits bill among multiple people
* 🔁 Repeat calculations in a loop
* ⚠️ Input validation for better user experience
* 🎯 Clean and formatted output

---

## 🛠️ Built With

* C++
* Standard Library (`iostream`, `iomanip`)
* Functions & Modular Programming
* Control Structures (`if/else`, `switch`, loops)

---
